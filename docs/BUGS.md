# BUGS - sanixdk.xyz

Canonical log of every bug surfaced during the audit. Each row must
mirror a `🟡` / `❌` / `⚪` row in
[`FEATURE_MATRIX.md`](./FEATURE_MATRIX.md).

Status values:

| Code | Meaning                                                          |
| ---- | ---------------------------------------------------------------- |
| 🔴   | Open, not yet fixed                                              |
| 🟠   | Fix in progress                                                  |
| 🟢   | Fix applied and verified by E2E                                  |
| ⚫   | Wontfix with reason (e.g. project policy / 3rd party limitation) |

Severity: `CRIT` (site unusable) · `HIGH` (feature visibly broken) ·
`MED` (polish / ux) · `LOW` (cosmetic / perf) · `NIT` (debatable).
